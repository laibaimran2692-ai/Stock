##  Project Overview
This project combines multiple stock datasets into a single unified dataset.  
Before merging, a **Source column** is added to each dataset to identify its origin.

The application provides advanced financial analysis and interactive visualizations using Streamlit and Plotly.

---

## Key Features

-  Multiple stock datasets merged into one
-  Source column added before merging
-  Interactive stock comparison
-  Date range filtering
-  Candlestick chart visualization
-  Moving averages (MA20 & MA50)
-  Daily return percentage analysis
-  Cumulative return comparison
-  Animated stock performance race
-  Futuristic animated UI (gradient + particles + neon effects)
